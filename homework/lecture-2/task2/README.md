# Resume Parser с GigaChat - Пример Structured Output

Приложение для демонстрации **Structured Output** с использованием GigaChat API, правильного парсинга JSON, валидации и работы с типобезопасными объектами Python.

## Описание проекта

Решение **Задания 2: Structured Output — парсинг ответа модели** с полной реализацией конвейера парсинга резюме:

1. **Интеграция с API** - отправляет запросы к GigaChat с инструкцией возвращать JSON
2. **Парсинг JSON** - извлекает и валидирует JSON из ответа модели
3. **Валидация типов** - использует Pydantic модели для гарантии целостности данных
4. **Форматированный вывод** - отображает результаты в читаемом структурированном формате

## Установка

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd resume-parser-gigachat

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Создать файл config.json с вашим API ключом
# Создайте файл config.json со следующим содержимым:
cat > config.json << 'EOF'
{
  "auth_key": "ВАШ_AUTH_KEY_ЗДЕСЬ"
}
EOF

# Замените "ВАШ_AUTH_KEY_ЗДЕСЬ" на ваш реальный ключ от GigaChat API
```

**Получение auth_key:**
1. Зарегистрируйтесь на [GigaChat](https://gigachat.devices.sberbank.ru)
2. Получите auth_key в личном кабинете
3. Вставьте его в config.json

## Использование

```bash
# Запустить парсер резюме
python main.py
```

## Детали реализации

### Данные модели (models.py)

Использует Pydantic для валидации:

```python
class Resume(BaseModel):
    name: str                      # Обязательно
    email: Optional[str]           # Опционально
    skills: List[str]              # Список навыков
    experience: List[Experience]   # История работы
    education: List[Education]     # Образовательный фон
```
